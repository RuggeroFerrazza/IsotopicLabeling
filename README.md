IsotopicLabelling R Package
======

# Installation

* Depends on
```
xcms, ecipex, stringr, gsubfn
```

* Install devtools
```
install.packages("devtools")
```

* Install 
```
library(devtools)
install_github("RuggeroFerrazza/IsotopicLabelling", build_vignettes = TRUE) 
```
